# Ape05_Automatass

Proyecto desarrollado para la gestión y simulación de autómatas utilizando Spring Boot en el backend y Angular en el frontend.

---

# Tecnologías utilizadas

## Backend
- Java
- Spring Boot
- Maven

## Frontend
- Angular
- TypeScript
- HTML
- CSS

---

# Requisitos

Instalar previamente:

## Backend
- Java JDK 17 o superior
- Maven

Verificar instalación:

```bash
java -version
mvn -version
```

---

## Frontend
- Node.js
- Angular CLI

Verificar instalación:

```bash
node -v
npm -v
ng version
```

---

# Clonar el proyecto

```bash
git clone [URL_DEL_REPOSITORIO](https://github.com/cesar050/APE05_Automatas.git)
```

Ingresar al proyecto:

```bash
cd Ape05_Automatass
```

---

# Ejecutar el Backend

Ingresar a la carpeta backend:

```bash
cd backend
```

Ejecutar Spring Boot:

```bash
mvn spring-boot:run
```

El backend se ejecutará en:

```bash
http://localhost:8080
```

---

# Ejecutar el Frontend

Abrir otra terminal e ingresar a frontend:

```bash
cd frontend
```

Instalar dependencias:

```bash
npm install
```

Ejecutar Angular:

```bash
ng serve
```

El frontend se ejecutará en:

```bash
http://localhost:4200
```

# Notas

